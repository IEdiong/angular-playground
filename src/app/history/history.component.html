This is my playground for learning Angular.

Some of the things that I have learnt so far include:

  • Components
    Learnt how to generate a component and all the parts of a component.
  • Directives
    Learnt about some directives eg, *ngFor, *ngIf, [(ngModel)] .
    Learnt that to use the [(ngModel)] you have to import FormsModule from @angular/core.
    Learnt how to bind a class (let's say the class is called 'select') to an element in the DOM using the syntax [class.select]='condition' to add a class when a condition is met.
  • Decorators
    Learnt about the @component and @injectable decorators.
  • Services
    Learnt how to use services in angular.
  • Dependency Injections
    Learnt how to use dependency injections in angular, service dependency injections in particular.
